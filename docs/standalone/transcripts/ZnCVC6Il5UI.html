The Payment Request API is a W3C standard that helps users share their payment and contact details with websites using web or native payment apps, like Google Pay.

If you are a payment app provider, we encourage you to explore the Payment Request APIs as a way to offer your users an enhanced experience on Chrome browsers. Payment app creators can use the web payment standard to deliver a better user experience. The Google Pay JavaScript Library leverages the Payment Request API to provide a native experience on Chrome for Android with a native experience rolling out soon on Chrome for desktop.
